# Como instalar o _NodeJS_

Baixando o fonte do _NodeJS_:
```bash
$ curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash -
```

Instalando o NodeJS:
```bash
$ sudo apt-get install -y nodejs
```

Após a instalação, verifique a versão do _NodeJS_:
```bash
$ node -v
```
